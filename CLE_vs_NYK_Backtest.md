## TC Formula Reference

```
TC PTS = pts × 0.85 (ACTIVE) / × 0.55 (Q) / × 0.0 (OUT)
TC REB = reb × 0.80
TC AST = ast × 0.75
TC 3PM = 3pm × 0.70
TC Line = TC Combined × 0.88
Edge = TC Combined − Line
Signal: OVER if edge > 3 | UNDER if edge < −3 | PASS otherwise
```
