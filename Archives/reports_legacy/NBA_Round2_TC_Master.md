## UPDATED TC FORMULA

```
TC_pts (playoffs) = pts × 0.90

PLAYOFF TOTAL = (Team_A_TC + Team_B_TC) × 1.18
Round to whole number

T-target = PLAYOFF TOTAL (whole number)

PICK TYPES:
  SPREAD  = whole number only, edge ≥ 3 pts
  TOTAL   = OVER if edge ≥ +8, UNDER if edge ≤ -5
  ML      = only if edge implies >60% win prob
  PROP    = edge ≥ 3 pts on conservative estimate
```
