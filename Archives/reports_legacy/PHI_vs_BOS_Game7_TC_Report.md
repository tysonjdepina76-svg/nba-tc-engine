## TC Formula Reference

```
TC PTS  = season_ppg × 0.85  (OUT → 0, Q → ×0.55)
TC REB  = season_reb × 0.80  (OUT → 0, Q → ×0.55)
TC AST  = season_ast × 0.80  (OUT → 0, Q → ×0.55)
TC 3PM  = season_3pm × 0.80  (OUT → 0, Q → ×0.55)
TC LINE = round(TC_PTS / 0.88)
EDGE    = TC_PTS − TC_LINE
```
