## Methodology

- **TC Combined** = sum of both teams' active player TC projections (stat × 0.85, Q × 0.55, OUT = 0)
- **TC Line** = TC Combined + 9.3 (NBA sport gap)
- **DK Total** = DraftKings over/under (from ESPN embedded odds)
- **Edge** = TC Line − DK Total
- **Signal:** OVER if Edge > +8.0, UNDER if Edge < −8.0, PASS otherwise
- **Hit:** Signal matches Actual result (PUSH counts as miss)
