# THE ONE FOOTBALL

A fast, free NFL prop projection + parlay builder app.

## Install and Run

1. Install Python 3 (if you don't have it).
2. Install requirements:

   pip install -r requirements.txt

3. Run the app:

   streamlit run app.py

Or upload all four files to GitHub and deploy to [Streamlit Cloud](https://streamlit.io/cloud).
