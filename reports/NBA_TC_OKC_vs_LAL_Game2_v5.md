## TC Formula Reference

```
TC PTS  = PTS × 0.85  (standard)
TC PTS  = PTS × 0.55  (⚠️ Questionable)
TC PTS  = 0            (❌ OUT)
TC LINE = TC PTS × 0.88  (rounded to whole number)
TC EDGE = TC PTS − TC LINE
T-target = whole number only
Min edge: +2 (legs), +3 (player props)
```
